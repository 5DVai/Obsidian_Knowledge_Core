Here's a curated "Tier S / Tier 1" map you can lean on as a coder and as a researcher.

==============

Tier S List - compressed

### Coding & Software Engineering — Tier S List

* MDN Web Docs -> [https://developer.mozilla.org/](https://developer.mozilla.org/) -> Web standards, HTML/CSS/JS reference
* GitHub -> [https://github.com/](https://github.com/) -> Real-world open-source code examples
* Stack Overflow -> [https://stackoverflow.com/](https://stackoverflow.com/) -> Debugging errors, practical coding fixes
* Python Documentation -> [https://docs.python.org/](https://docs.python.org/) -> Official Python language and library
* React Docs -> [https://react.dev/](https://react.dev/) -> Official React components and patterns
* TypeScript Docs -> [https://www.typescriptlang.org/docs/](https://www.typescriptlang.org/docs/) -> TypeScript types, compiler, config reference
* Next.js Docs -> [https://nextjs.org/docs](https://nextjs.org/docs) -> Next.js routing, data, deployment

MDN Web Docs – Canonical reference for open-web technologies: HTML, CSS, JavaScript, browser APIs, HTTP, WebAssembly, PWA patterns, plus solid introductory learning tracks.
GitHub – Largest open-source code host and de-facto archive of "how things are actually built." Essential for reading real-world implementations, issues, discussions, and docs.
Stack Overflow – Massive Q&A archive for practical coding issues, error messages, edge cases, and idioms in almost every language and framework.

These are the ground truth for behavior, APIs, and version-specific changes. Treat them as Tier S for whatever stack you are in.

Tier 1 List
DevDocs – https://devdocs.io/
Aggregates hundreds of official docs (languages, frameworks, libraries) into a unified, fast, offline-friendly interface with instant search.
freeCodeCamp – https://www.freecodecamp.org/
Project-based curriculum for web dev, JavaScript, data structures & algorithms, and some data/ML content, with an emphasis on hands-on coding and cert paths.
Sourcegraph – https://sourcegraph.com/
Code search and cross-repo navigation for large codebases; great for pattern-hunting, reading OSS at scale, and understanding how APIs are used in the wild.

### Multidisciplinary Scholarly Search & Preprints — Tier S List

* arXiv -> [https://arxiv.org/](https://arxiv.org/) -> Latest preprints in STEM fields
* Google Scholar -> [https://scholar.google.com/](https://scholar.google.com/) -> Cross-publisher scholarly article discovery
* JSTOR -> [https://www.jstor.org/](https://www.jstor.org/) -> Archived journals, books, primary sources
* Semantic Scholar -> [https://www.semanticscholar.org/](https://www.semanticscholar.org/) -> AI-assisted paper search and summaries
* DOAJ -> [https://doaj.org/](https://doaj.org/) -> Find peer-reviewed open-access journals

arXiv – Core preprint server for physics, math, CS, quantitative finance, stats, and more; millions of open-access manuscripts, often the first place cutting-edge work appears.
Google Scholar – Broad academic search engine across publishers, preprint servers, theses, courts, etc. Good "first stop" index for most scholarly domains.
JSTOR – Non-profit digital library with millions of journal articles, books, and primary sources across ~75 disciplines, especially strong in humanities and social sciences.
Semantic Scholar – AI-powered research tool with semantic search, citation graphs, and paper summaries across scientific domains; particularly good for CS and AI.
Directory of Open Access Journals (DOAJ) – Curated index of peer-reviewed open-access journals across science, tech, medicine, social sciences, and humanities—excellent for free, legitimate research literature.

These are the ground truth for behavior, APIs, and version-specific changes. Treat them as Tier S for whatever stack you are in.

Tier 1 List
SSRN (Social Science Research Network) – https://www.ssrn.com/
Major preprint and working-paper repository for economics, law, finance, and social sciences, now expanding into other fields.
ScienceDirect (Elsevier) – https://www.sciencedirect.com/
Huge database of scientific, technical, and medical journals and books; often paywalled but central in many STEM fields.

### Mathematics — Tier S List

* Wolfram MathWorld -> [https://mathworld.wolfram.com/](https://mathworld.wolfram.com/) -> Definitions, theorems, formulas, visualizations
* OEIS -> [https://oeis.org/](https://oeis.org/) -> Identify and explore integer sequences
* MIT OCW – Mathematics -> [https://ocw.mit.edu/courses/mathematics/](https://ocw.mit.edu/courses/mathematics/) -> Full math courses, notes, problems
* arXiv – Math -> [https://arxiv.org/archive/math](https://arxiv.org/archive/math) -> Research preprints in modern mathematics

Wolfram MathWorld – Comprehensive online math encyclopedia with definitions, theorems, formulas, and illustrations across pure and applied math.
OEIS – On-Line Encyclopedia of Integer Sequences – Canonical database of integer sequences; invaluable for combinatorics, number theory, and "what sequence is this?" identifications.
MIT OpenCourseWare – Mathematics – Full course materials (lectures, notes, problem sets) for MIT math courses—from intro calculus to grad-level topics—free and open license.
MIT Open Learning
arXiv (Math categories) – Primary preprint feed for modern math: algebra, analysis, geometry, probability, etc. Same platform as above, focused on math.

These are the ground truth for behavior, APIs, and version-specific changes. Treat them as Tier S for whatever stack you are in.

Tier 1 List
PlanetMath / other math encyclopedias – https://planetmath.org/
Community-driven math encyclopedia; less polished than MathWorld but still useful for alternative explanations and some niche topics.

### Science, Engineering & Medicine — Tier S List

* PubMed -> [https://pubmed.ncbi.nlm.nih.gov/](https://pubmed.ncbi.nlm.nih.gov/) -> Biomedicine abstracts, trials, reviews
* NASA ADS -> [https://ui.adsabs.harvard.edu/](https://ui.adsabs.harvard.edu/) -> Astrophysics and astronomy research literature
* NASA Science -> [https://science.nasa.gov/](https://science.nasa.gov/) -> Authoritative space and Earth science
* ScienceDirect -> [https://www.sciencedirect.com/](https://www.sciencedirect.com/) -> Engineering, physics, chemistry journal articles
* PubChem -> [https://pubchem.ncbi.nlm.nih.gov/](https://pubchem.ncbi.nlm.nih.gov/) -> Chemical properties, structures, bioactivities

PubMed – Premier database for biomedical and life-sciences literature, indexing tens of millions of citations and linking to full text where available.
NASA ADS (Astrophysics Data System) – (Bonus for astro/cosmo) Specialized index for astronomy and astrophysics literature, widely used by researchers.
NASA Science / Earth & Space – Authoritative source on space, Earth science, and missions, with data, imagery, and accessible explainers for both public and researchers.
ScienceDirect – Broad STM (science, technical, medical) platform: physics, engineering, chemistry, materials, environmental science, and more.

These are the ground truth for behavior, APIs, and version-specific changes. Treat them as Tier S for whatever stack you are in.

Tier 1 List
Semantic Scholar (STEM focus) – https://www.semanticscholar.org/
Very strong for CS, AI, and many STEM subfields; semantic filters help surface influential work and related papers efficiently.
DOAJ (STEM journals) – https://doaj.org/
To locate open-access journals in niche scientific or engineering areas when you want legally free full-text.

### History, Philosophy & Humanities — Tier S List

* Stanford Encyclopedia of Philosophy -> [https://plato.stanford.edu/](https://plato.stanford.edu/) -> Expert, citable philosophy reference articles
* Internet History Sourcebooks -> [https://sourcebooks.fordham.edu/](https://sourcebooks.fordham.edu/) -> Curated historical primary source collections
* Project Gutenberg -> [https://www.gutenberg.org/](https://www.gutenberg.org/) -> Public-domain classics and historical texts
* Perseus Digital Library -> [https://www.perseus.tufts.edu/](https://www.perseus.tufts.edu/) -> Greek, Roman texts with tools
* JSTOR (Humanities) -> [https://www.jstor.org/](https://www.jstor.org/) -> Scholarly humanities and history journals

Stanford Encyclopedia of Philosophy (SEP) – Peer-reviewed, continuously updated reference for philosophy. Articles are written and maintained by domain experts and are often citable in academic work.
Internet History Sourcebooks Project (Fordham) – Curated collections of public-domain and copy-permitted primary sources for ancient, medieval, and modern history; widely used as an online textbook replacement.
Project Gutenberg – Massive library of 60k+ public-domain books—classics in literature, philosophy, and historical texts—free to read and download.
Perseus Digital Library (Tufts) – Digital library focused on classical Greek and Roman texts (plus other collections), with aligned translations, lexical tools, and historical context.
JSTOR (Humanities & History) – Core journal and book archive for history, literature, classics, religious studies, and other humanities fields.

These are the ground truth for behavior, APIs, and version-specific changes. Treat them as Tier S for whatever stack you are in.

Tier 1 List
Internet Archive (Texts) – https://archive.org/details/texts
Huge repository of scanned books and documents (many historical and scholarly) with full-text search; quality varies but coverage is unmatched.
